# Cotor 업그레이드 가이드

원문: [UPGRADE_GUIDE.md](UPGRADE_GUIDE.md)

## Cotor 1.0 이후 무엇이 달라졌는가

이 문서는 초기 버전에서 개선된 주요 포인트를 설명합니다.

### 주요 개선

- 향상된 CLI 명령 체계
- 실시간 진행 모니터링
- 파이프라인 검증
- 사용자 친화적 오류 메시지
- dry-run 모드

### 기술 개선

- 신규 의존성
- 신규 컴포넌트
- 조건/루프 파이프라인
- 결과 인텔리전스
- 이벤트 시스템 업데이트

## 마이그레이션 가이드

원문은 다음 순서를 권장합니다.

1. 기존 스크립트를 새 명령 체계에 맞게 수정
2. 실행 전 validation 추가
3. 디버깅에 verbose 모드 사용
4. dry-run으로 변경 검증

## 추가 항목

- 성능 향상
- 보안 강화
- 새 문서
- 테스트
- 로드맵
- 지원 / 감사

실제 예제 출력과 이전/이후 스크립트 비교는 원문 문서를 보십시오.
