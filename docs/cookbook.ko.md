# Cotor Cookbook: 실전 시나리오 파이프라인 모음

원문: [cookbook.md](cookbook.md)

이 문서는 현재 원문도 한국어 중심으로 작성된 예제 모음입니다. `.ko.md` 경로를 통해 한국어 동반 문서를 명시적으로 제공합니다.

## 포함 시나리오

- 코드 리뷰 자동화
- 레포 전체 리팩토링 오케스트레이션
- 테스트 자동 생성 + 품질 보정 루프
- 대량 파일 처리
- 빠른 시작 / 스모크 체크
- 패턴 선택 가이드

## 사용법

각 시나리오의 실행 커맨드와 실무 팁은 원문 문서를 그대로 따라가면 됩니다.
