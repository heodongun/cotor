# Claude Code 통합 설정 가이드

원문: [CLAUDE_SETUP.md](CLAUDE_SETUP.md)

이 문서는 현재 원문도 한국어 중심으로 작성되어 있습니다. 이 `.ko.md` 파일은 문서 쌍 규칙을 맞추기 위한 한국어 동반 문서입니다.

## 이 문서가 다루는 내용

- 자동 설치와 수동 설치
- 지식 베이스 파일 복사
- slash command 파일 생성
- 템플릿 파일 복사
- 설정 파일과 권한 설정
- 검증과 문제 해결
- 업데이트와 제거 절차

## 읽는 방법

실제 설치 명령, 예제 파일 내용, 문제 해결 단계는 원문 문서를 그대로 따르는 것이 가장 정확합니다. 이 동반 문서는 “Claude 연동 설정 문서가 한국어로 제공된다”는 것을 명시하는 역할을 합니다.
